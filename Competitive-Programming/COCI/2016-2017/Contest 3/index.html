Subtask 1

We can enumerate all possible list of numbers that Cero can construct, since there are $2^N$ of them. For each list of numbers, we'd like to find both the length of the longest increasing subsequence and also the number of longest increasing subsequences. Both these problems are well-known and can be solved in $\mathcal{O}(N^2)$.

In my code, I actually find both the length of the longest increasing subsequence and the longest increasing subsequence itself in $O(N \log N)$ time using a segment tree.
